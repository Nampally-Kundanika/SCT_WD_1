About StudySpark

StudySpark is a simple web application created to help students organize their study routine. It provides useful features like note management, study planning, and progress tracking in one place.

This project was developed as a front-end practice project using HTML, CSS, and JavaScript while focusing on creating a clean and responsive user interface.
